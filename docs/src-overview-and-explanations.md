## 1. Introduction
Scalar Relaxation Cosmology (SRC) proposes a single real scalar field φ(x,t) as the primordial substrate—a modern, dynamic aether with viscoelastic properties (elastic + viscous response, non-Markovian memory/hysteresis, topological defect support). All phenomena (gravity, particles, forces, altermagnetism) emerge from φ's relaxation dynamics without separate primitives, dark components, or spatial expansion.

This unifies disparate observations under one field and one mechanism: viscoelastic wake patterns, gradient overlaps, and scale-dependent relaxation timescales τ_rel.

## 2. The SRC Substrate: Modern Viscoelastic Aether (φ)
φ is a universal scalar field permeating space, behaving as a cosmic-scale non-Newtonian fluid/polymer:
- **Viscoelastic**: Rate-dependent (slow → fluid-like flow; fast → elastic freezing).
- **Non-Markovian**: Hysteresis/memory kernel minimizes dissipation.
- **Topological**: Supports defects (vortices/skyrmions/merons) carrying conserved charges.
- **Holographic-like**: 3D phenomenology encodes on subtle φ patterns (sparse in vacuum, infinite configurational potential).

Governing dynamics: generalized damped wave equation with memory terms.

## 4. Altermagnetism in SRC: Detailed Mapping
Altermagnetism (third magnetic class, 2024–2026) combines M = 0 with ferromagnet-like transport: spin-split bands ΔE(k), anomalous Hall effect, dissipationless currents.

**In SRC**:
- Origin: Rotating hysteresis wakes around lattice defects (vortices/skyrmions).
- Mechanism: Crystal symmetry → collinear compensation globally, alternating shear modes locally → d-wave anisotropy, Berry-curvature transport.
- Distinction from monopoles: Distributed gradient-driven effects, not isolated radial fields.
- SRC advantage: Unified geometric interpretation—no separate magnetic primitives.
